
# 🚀 ImagePixel – Full Stack Image Management Platform

ImagePixel is a modern full-stack web application that allows users to upload, manage, and explore images with secure authentication, fast performance, and a clean UI.

---

## ✨ Features

* 🔐 User Authentication (Signup/Login with NextAuth)
* 📸 Image Upload & Management (ImageKit integration)
* 🗂️ Dynamic Product/Image Pages
* 💳 Secure Payments using Razorpay
* 📧 Email Notifications with Nodemailer
* 🌐 REST APIs using Next.js App Router
* 🧠 MongoDB Database Integration with Mongoose
* 🎨 Modern UI using Tailwind CSS + DaisyUI

---

## 🛠️ Tech Stack

### Frontend

* Next.js 15
* React 18
* Tailwind CSS
* DaisyUI

### Backend

* Next.js API Routes

### Database

* MongoDB + Mongoose

### Authentication

* NextAuth

### Other Integrations

* ImageKit (Image Storage)
* Razorpay (Payments)
* Nodemailer (Email Service)

## 📁 Project Structure

```bash
app/              # App Router pages
```

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/nageshkumar74/PhotoShop.git
cd PhotoShop
```

---

### 2️⃣ Install dependencies

```bash
npm install
```

---

### 3️⃣ Setup environment variables

Create a `.env.local` file and add:

```env
MONGODB_URI=your_mongodb_connection
NEXTAUTH_SECRET=your_secret
NEXTAUTH_URL=http://localhost:3000

IMAGEKIT_PUBLIC_KEY=your_key
IMAGEKIT_PRIVATE_KEY=your_key
IMAGEKIT_URL_ENDPOINT=your_url

RAZORPAY_KEY_ID=your_key
RAZORPAY_KEY_SECRET=your_secret

EMAIL_USER=your_email
EMAIL_PASS=your_password
```

---

### 4️⃣ Run the project

```bash
npm run dev
```

👉 Open: http://localhost:3000
